Twitter Clone

A full stack clone of twitter. I used Node.js, Express, and PostgreSQL for the server side handling, and React on the front end. This site allows users to post tweets to share with other users as well as comment on others posts. All of this data is being stored on a PostgreSQL database in a Heroku server. Please feel free to make an account and check it out!

Deployed Site | Github




Pet Rock Store

A full stack e-commerce website built using React for the front end, and Express, Node.js, and PostgreSQL for the back end. Passwords are hashed when stored in the database to ensure no plain text passwords are accessible. Admin panel accessibility enabled only for authorized users.

Deployed Site | GitHub
